# Список изменений

## [1.0.1] - 2026-03-23

### Исправлено
- Исправлена критическая ошибка инициализации `Sabre\DAV\Client` на чистых установках WordPress.
- Решена проблема отсутствия зависимости `psr/log` в сборке плагина.

### Изменено
- Кастомный автозагрузчик заменен на стандартный `vendor/autoload.php`.
- Обновлен скрипт сборки для включения всех необходимых зависимостей.
- Папка `releases/` полностью исключена из репозитория.

## [1.0.0] - 2026-03-10

### Добавлено
- Начальный форк от `external-files-from-webdav`.
- Полный ребрендинг проекта.
- Выбор пресетов для Nextcloud и Яндекс.Диска.
- Кнопка проверки соединения (AJAX).
- Интеллектуальное кэширование и проксирование превью изображений.
- Уникальные иконки для различных провайдеров.

### Изменено
- Глобальное переименование всех функций на `wwml_`.
- Пространство имен обновлено до `KiSa\WebDavMediaLibrary`.
