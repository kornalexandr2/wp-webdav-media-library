# WP WebDav Media Library

Полноценный самостоятельный плагин для интеграции WebDAV в медиабиблиотеку WordPress.

Этот плагин позволяет подключать внешние облачные хранилища (Nextcloud, Яндекс.Диск и др.) напрямую к вашему сайту. Файлы остаются в облаке, но отображаются и работают в WordPress как родные медиафайлы.

## Особенности

- **Без физического импорта:** Файлы не занимают место на вашем сервере.
- **Нативная интеграция:** В окне выбора медиафайлов появляется вкладка "WebDAV Library".
- **Проксирование ссылок:** Ссылки на файлы привязаны к вашему домену (настраивается в настройках).
- **Умные пресеты:** Быстрая настройка для Nextcloud и Яндекс.Диска.
- **Кэширование превью:** Быстрое отображение миниатюр без лишней нагрузки на облако.

## Требования

- PHP 8.1+
- WordPress 6.2+

## Установка

1. Загрузите папку с плагином в директорию `/wp-content/plugins/` вашего сайта.
2. Активируйте плагин через раздел «Плагины» в консоли WordPress.
3. Перейдите в **Настройки -> WebDAV Media**, чтобы указать данные для подключения.
