# WP WebDav Media Library

Независимый форк проекта [external-files-from-webdav](https://github.com/threadi/external-files-from-webdav).

Этот плагин расширяет возможности медиабиблиотеки WordPress, добавляя поддержку WebDAV в качестве источника внешних файлов.

## Особенности

- Подключение к любому серверу WebDAV (Nextcloud, Яндекс.Диск и др.).
- Использование внешних файлов без их загрузки на ваш сервер.
- Поддержка учетных данных для нескольких пользователей (настройки в профиле пользователя).
- Улучшенный интерфейс выбора провайдеров.
- Локальное кэширование превью изображений для повышения производительности.

## Требования

- PHP 8.1+
- WordPress 6.2+
- [External files in Media Library](https://wordpress.org/plugins/external-files-in-media-library/) (Рекомендуемый родительский плагин, хотя этот форк может работать независимо в будущем).

## Установка

1. Склонируйте или загрузите репозиторий в папку `wp-content/plugins`.
2. Выполните `composer install` для установки зависимостей (Sabre/DAV).
3. Активируйте плагин в WordPress.
