Полноценная интеграция WebDAV в медиабиблиотеку WordPress. Файлы остаются в облаке, но работают как родные.

== Description ==

WP WebDav Media Library — это полностью самостоятельный плагин, который добавляет поддержку внешних хранилищ WebDAV (таких как Nextcloud, Яндекс.Диск, ownCloud и другие) прямо в интерфейс WordPress.

В отличие от других решений, этот плагин не копирует файлы на ваш сервер. Он создает «виртуальные» медиафайлы, ссылки на которые проксируются через ваш сайт. Это экономит дисковое пространство и упрощает работу с большими библиотеками данных.

Основные возможности:
* **Нативная вкладка "WebDAV Library"** в стандартном окне выбора медиафайлов.
* **Настраиваемый прокси-префикс** для внешних ссылок.
* **Поддержка пресетов** для популярных сервисов.
* **Проверка подключения** в один клик.
* **Кэширование миниатюр** для мгновенной загрузки.

== Installation ==

1. Загрузите архив с плагином в разделе «Плагины» -> «Добавить новый» -> «Загрузить плагин».
2. Активируйте плагин.
3. Перейдите в раздел «Настройки» -> «WebDAV Media» и введите данные вашего сервера.
